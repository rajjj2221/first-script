# Sistema de Cadastro de Notas Escolares

Este é o meu primeiro script funcional após 3 dias estudando Python.

## Funcionalidades
- Cadastrar alunos com nome, notas (matemática, português, ciências) e turma
- Mostrar lista de todos os alunos cadastrados
- Buscar aluno específico e mostrar suas notas
- Calcular a média da turma
